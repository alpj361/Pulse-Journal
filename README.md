# Pulse Journal

Repositorio principal del proyecto Pulse Journal - un sistema completo de procesamiento de noticias y contenido digital.

## Estructura del Proyecto

### Componentes Principales

- **04bc0317-b8c9-4395-93f8-baaf4706af5c**: Aplicación móvil web
- **ExtractorT**: Extractor y procesador de contenido de Twitter/X
- **ExtractorW**: Extractor y procesador de contenido web
- **ThePulse**: Aplicación principal y frontend
- **NewsCron**: Sistema de automatización y procesamiento de noticias
- **WebAgent**: Agente web para automatización
- **LauraMemoryService**: Servicio de gestión de memoria y contexto
- **mcp-chrome**: Integración con Chrome para automatización
